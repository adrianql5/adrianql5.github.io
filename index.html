Proyectos

Ixildu-hasiko-da!

Proyecto de Bases de Datos II: Sistema de gestión para un cine. Incluye funcionalidades para la gestión de películas, sesiones, reservas y usuarios, implementando consultas SQL avanzadas y una interfaz de usuario intuitiva.

SQL Java Bases de Datos

Características destacadas

  • Gestión integral de reservas y sesiones
  • Consultas optimizadas
  • Interfaz amigable

Dominivm

Proyecto académico de Diseño de Software: prototipo de videojuego de estrategia por turnos ambientado en civilizaciones de la Edad Antigua. Enfocado en el diseño arquitectónico y patrones de software aplicados con UML y StarUML.

UML Patrones de diseño StarUML Diseño de Software

Elementos de diseño

  • Jerarquías de clases
  • Diagrama de componentes e interacciones
  • Aplicación de patrones de software

Monopoly en Java

Implementación digital del clásico juego de mesa Monopoly, desarrollada en Java. Incluye gestión de turnos, lógica de propiedades, tarjetas y banca, así como una interfaz gráfica para la interacción de los jugadores.

Java POO Interfaz Gráfica

Características clave

  • Gestión de reglas y eventos del juego
  • Interfaz interactiva
  • Lógica de IA básica para jugadores

ShadersSea

Simulación interactiva de un faro en una isla que se defiende de oleadas de barcos enemigos. El faro lanza proyectiles a los barcos, incrementando la dificultad con cada oleada superada. Proyecto centrado en la programación gráfica y simulación de física.

OpenGL C++ Simulación Gráfica

Puntos destacados

  • Sistema de oleadas y dificultad incremental
  • Física de colisiones y proyectiles
  • Gráficos en tiempo real
